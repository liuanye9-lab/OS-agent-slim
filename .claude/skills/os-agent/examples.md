# OS Agent 使用示例

## 示例 1：诊断项目架构

```
/os-agent 分析当前项目架构，找出潜在问题
```

Dashboard 将显示：
- Agent 理解意图 → 计算预算 → 查找记忆 → 搜索资料 → 构建上下文
- 进度百分比实时更新
- 决策卡片解释"为什么选择这个架构模式"

## 示例 2：优化代码质量

```
/os-agent 扫描代码库，找出性能瓶颈和可优化点
```

## 示例 3：SkillOpt 自优化

```
/os-agent 基于最近的任务反馈，运行一轮 SkillOpt 自优化
```

## 示例 4：生成开发提示词

```
/os-agent 为这个功能生成可直接执行的 Codex 开发提示词
```
